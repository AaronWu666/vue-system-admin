## 克隆项目
```
git clone https://github.com/FTLLOVE/vue-system-admin.git
```

## 依赖安装
```
yarn install or npm install
```

## 编译启动项目
```
yarn serve or npm serve
```

## 打包部署
```
yarn build
```
